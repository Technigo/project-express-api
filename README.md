# Express API Project

A book API build in Node using Express.

Available endpoints:

- Root: /
- Books: /books
- Book by id: /books/:id

## Learning Objectives

- How to install and set up Express server
- How to create rotes
- Practice data manipulation in JavaScript - filtering and limiting arrays
- How to deploy Node projects

## Tech

- Heroku
- Postman
- Express
- JavaScript
