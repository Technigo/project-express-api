# Express API Project

This is a book API. By using different endpoints you can find the information you're looking for. 

/books
/books/page/:page 
/books/id/:id
/books/:title
/books/authors/:authors
/books/rating/:rating

I used slice() so page shows 30 books/page.

## Tech
JS
Node.js
Express
