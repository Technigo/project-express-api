Custom Netflix movie API

GET endpoints

  • Get all movies

    e.g. GET /api/movies

  • Get a specific movie

    e.g. GET /api/movies/60020826

  • Get all movies with a specific actor

    e.g. GET /api/movies?actor=Anna

  • Get all movies for a specific year

    e.g. GET /api/movies?year=2017

  • Get all movies with a specific duration (min)

    e.g. GET /api/movies?duration=105

  • Get all movies with pagination

    e.g. GET /api/movies?page=1

POST endpoint

  • Create a new movie

    e.g. POST /api/movies with payload:

    Object validates correctly

                { 
  "title": "Microsoft Horror", 
  "director": "Bill gates", 
  "cast": "Val Kilmer, Tom Cruise",
  "country": "USA",
  "date_added": "January 28, 2020",
  "release_year": 2020,
  "rating": "TV-20",
  "duration": "120 min",
  "listed_in": "Horror movies",
  "description": "To big to describe in a few words.",
  "type": "Movie" 
}

    Object validates incorrectly

                { 
  "title": "M",
  "director": "Bill gates", 
  "cast": "Val Kilmer, Tom Cruise",
  "country": "USA",
  "date_added": "January 28, 2020",
  "release_year": 2020,
  "rating": "",
  "duration": "120 min",
  "listed_in": "",
  "description": "To big to describe in a few words.",
  "type": "Movie" 
}

PUT endpoint

  • Update a specific movie

    e.g. PUT /api/movies/:id with payload: 

                  {
  name: 'Fredrik' 
}

DELETE endpoint

  • Delete a specific movie

    e.g. DELETE /api/movies/80233258
